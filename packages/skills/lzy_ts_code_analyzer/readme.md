# 一组用于 TS 代码分析的skills

## skills
Skill             | 说明
------------------------------------------------------------------------------------------------
architecture-analysis | 全面的前端架构分析器，用于识别技术栈、构建工具和架构模式。
                  | 支持Vue/React/Angular框架、Node.js环境、包管理器、TypeScript使用、代码检查工具和架构模式的分析，
                  | 提供多种输出格式包括执行摘要和可视化。
------------------------------------------------------------------------------------------------
ts-morph-analyzer  | 调试TypeScript/JavaScript错误时使用，通过追踪调用链快速理解代码库、进行架构决策或代码质量审查。
                  | 可提取函数签名和JSDoc、追踪调用层次、检测代码异味并跟踪数据流。
------------------------------------------------------------------------------------------------
coding-standards   | 适用于TypeScript、JavaScript、React和Node.js开发的通用编码标准、最佳实践和模式。
------------------------------------------------------------------------------------------------
react-patterns     | 现代React模式和原则，涵盖Hooks、组合、性能优化和TypeScript最佳实践。
------------------------------------------------------------------------------------------------
typescript-expert  | 精通 TypeScript 和 JavaScript，在类型层级编程、性能优化、monorepo 管理、迁移策略和现代工具方面拥有深厚的知识。
                  | 主动使用 PROACTIVELY 解决任何 TypeScript/JavaScript 问题，包括复杂的类型体操、构建性能。
------------------------------------------------------------------------------------------------
frontend-mobile-security-xss-scan | 前端安全专家，专注于跨站脚本（XSS）漏洞检测和预防。
                  | 分析React、Vue、Angular和原生JavaScript代码以识别注入点。
------------------------------------------------------------------------------------------------
audit-context-building | 实现超细粒度的逐行代码分析，在发现漏洞或错误之前构建深入的架构上下文。
------------------------------------------------------------------------------------------------
